# 🎬Square Eyes 

A modern and responsive streaming website where users can explore movies, view details, and enjoy an engaging UI experience.
This project was built as part of frontend development for JavaScript1 and Cross Course Project at Noroff.

## 🔧 Getting Started

### Installation
1. Clone the repository 
```bash
git clone https://github.com/meluhrose/SquareEyes.git
```
2. Open the project folder

```bash
cd SquareEyes
```

3. Open index.html in your browser
  - Or run a local server using VS Code Live Server
